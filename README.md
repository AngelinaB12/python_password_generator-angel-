# Python Password Generator

A simple Python project that generates secure random passwords using Python.

## Features
- Generate strong random passwords
- Includes letters, numbers, and symbols
- User chooses password length

## Technologies Used
- Python
- Random module
- String module

## How to Run

1. Install Python
2. Run the script:

python password_generator.py

## Example Output


Enter password length: 10
Generated Password: A7#kP2@zQ
